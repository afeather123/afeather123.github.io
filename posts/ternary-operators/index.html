A ternary operator is an expression that exists in most programming languages that takes 3 arguments; they look like this:

conditional_statement ? true_result : false_result;

Example:

let burner_on = true let stove_temp = burner_on === true ? ‘The stove is hot’ : ‘The stove is cool’ console.log(stove_temp) // This will print out ‘The stove is hot’

Ternary operators have a conditional statement, a ‘true’ result, and a ‘false’ result. It then returns either the true or false result based on whether the condition resolves to true or false.

Ternary operators are often compared to a simple if-else statement and this is mostly a good comparision. A more percise analogy for what a ternary operator is, is a function that looks like this:

function TernaryOperator(condition, true_result, false_result) { if(condition) { return true_result } else { return false_result } }

This concept of returning a different value based on a single boolean condition is so common that it makes a lot of sense to have a universal shorthand for it.
